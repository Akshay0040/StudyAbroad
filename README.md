##  Features

* Browse study abroad programs
* View detailed program information
* Ratings and statistics display
* Program details (duration, fees, language, deadline)
* Direct link to official university websites
* Fully responsive for all mobile screen sizes
* Works on both Android and iOS

---

##  Tech Stack

* React Native CLI
* TypeScript
* React Navigation (Native Stack)
* Custom Hooks
* Animated API
* Modular Component Architecture

---

##  Installation

1. Clone the repository:

```
git clone <https://github.com/Akshay0040/StudyAbroad.git>
```

2. Install dependencies:

```
npm install
```

---

##  Run the App

### Android

```
npx react-native run-android
```

### iOS

```
npx pod-install
npx react-native run-ios
```

##  Notes

* Ensure Metro bundler is running before launching the app
* For iOS, make sure Xcode and CocoaPods are installed
* Reset cache if facing issues:

```
npx react-native start --reset-cache
```
